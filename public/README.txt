
PNG favicon pack for SHILL / ChillMeOut
=======================================

Place all files in your Vite project's /public directory:

  /public/favicon-16x16.png
  /public/favicon-32x32.png
  /public/apple-touch-icon.png
  /public/android-chrome-192x192.png
  /public/android-chrome-512x512.png
  /public/site.webmanifest

Then add these tags inside <head> in index.html:

  <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
  <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
  <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
  <link rel="manifest" href="/site.webmanifest">
  <meta name="theme-color" content="#0b1020">

Most modern browsers accept PNG favicons. If you still need .ico, I can export one on request.
